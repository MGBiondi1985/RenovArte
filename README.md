# RenovArte - Página Web de Peluquería

Este repositorio contiene el código fuente de la página web de la peluquería **RenovArte**, un espacio donde la creatividad y el arte de la peluquería se unen para ofrecer servicios personalizados y de calidad.

## Descripción

La página web está diseñada para mostrar los servicios que ofrece RenovArte, así como información sobre la historia del salón y cómo agendar turnos. La web es completamente responsiva y ofrece una interfaz amigable para usuarios en dispositivos móviles y de escritorio.

### Secciones Principales:
- **Nosotros**: Historia del salón y los estilistas.
- **Servicios**: Lista de los servicios disponibles (corte, tintura, masajes, entre otros).
- **Turnos**: Sistema de agendado de turnos en línea, accesible mediante una nueva pestaña.
- **Testimonios**: Opiniones de clientes satisfechos.
- **Galería**: Imágenes de trabajos realizados por RenovArte.

## Tecnologías Utilizadas

- **HTML5**: Para la estructura de la página web.
- **CSS3**: Para el diseño y el estilo visual, incluyendo la fuente *Satisfy*.
- **JavaScript**: Para las funcionalidades dinámicas y de interacción.
- **Google Fonts**: Para integrar la fuente `Satisfy` en el título de la página.
- **Visual Studio Code**: Editor utilizado para el desarrollo.
- **XAMPP**: Entorno de desarrollo local para pruebas.
